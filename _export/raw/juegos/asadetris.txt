====== Asadetris ======

Asadetris es un videojuego similar a Tetris.


Todo el escenario está representado por el estómago de una
persona, las piezas son porciones de comida que usted tiene
que acomodar correctamente.

{{:juegos:asadetris:asadetris_3.png|}}


===== Sobre el desarrollo =====


Asadetris es un juego desarrollado en equipo, un grupo de
personas con ganas de hacer un juego divertido, libre y que
les permita aprender del desarrollo.

Este grupo está abierto a sugerencias y nuevos participantes,
vea los mensajes de foro mas abajo para obtener mas detalles.


===== Código del proyecto =====


Como este proyecto está en desarrollo y es grupal, contamos
con un repositorio de código en común. Ahí publicamos el código
fuente del juego y todos los cambios que realizamos.

Para obtener una copia del código necesita el software "Mercurial"
y luego escribir el siguiente comando:

<code>
hg clone https://asadetris.googlecode.com/hg/ asadetris
</code>

También puedes suscribirte a la lista de correo "alerts" para
recibir un correo de notificación cada vez que alguien actualiza
el repositorio:

  * [[http://groups.google.es/group/losersjuegos-alerts]]
