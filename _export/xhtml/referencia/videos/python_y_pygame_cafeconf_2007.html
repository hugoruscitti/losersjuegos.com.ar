Como desarrollar juegos con Python y Pygame

El motivo de la charla es brindar una aproximación al desarrollo de videojuegos utilizando el lenguaje de programación Python y la biblioteca Pygame.

El video abarca temas como “¿Por qué hacer videojuegos?”, “¿Que es Python y pygame?” … hasta el desarrollo de un videojuego paso a paso.

La charla tuvo lugar en el evento de software libre CaFeCONF 2007 organizado por el grupo de usuarios CaFeLUG en la ciudad de Buenos Aires (Argentina).
