Sistema personalizado para realizar videojuegos

por hugoruscitti » Mié Dic 12, 2007 4:23 pm

Saludos a todos, quería comentarles que hace algunos días preparé una
distribución basada en Ubuntu 7.10 [1] que incluye muchas herramientas para
desarrollar videojuegos.

Pienso que este CD puede ser de utilidad para todas aquellas personas que
recién comienzan y tienen algunas dificultades para instalar GNU/Linux,
SDL, python y muchas otras herramientas de desarrollo. Además, este CD puede
ser de utilidad para difundir el desarrollo de juegos en las charlas;
casualmente nosotros en CaFeCONF 2007 regalamos CD con una versión anterior de
este sistema.

Pueden obtener mas información del CD en las siguientes direcciones:

Información general y descargas: http://www.losersjuegos.com.ar/utilidad ... a.t2t.html
Preguntas frecuentes: http://www.losersjuegos.com.ar/utilidad ... q.t2t.html




El sistema se puede descargar en formato imagen de CD (.iso), y una vez
descargado se puede transferir a un CDROM. El disco creado será un LiveCD [2]
por lo que podrán probarlo en sus equipos sin instalar ni modificar dada. El
sistema operativo funciona directamente desde el CD, y si quieren, luego se
puede instalar en el disco duro del equipo.

Me gustaría que prueben el sistema y comenten que les parece, actualmente
tiene instaladas todas las bibliotecas que considero interesantes para hacer
juegos, como SDL, pygame, panda3d, rubygame, soya3d ...

Pienso que a futuro podré escribir algunos tutoriales sobre lo que incluye el
CD. ¿Alguien me podrá ayudar?.

Aprovecho este mensaje para agradecer a William Javier Del Valle Meza, por
brindarnos espacio para alojar la imagen de este CD. ¡Gracias amigo!.

[1] - http://es.wikipedia.org/wiki/Ubuntu_%28 ... n_Linux%29
[2] - http://es.wikipedia.org/wiki/LiveCD

Muchas gracias.
Hugo Ruscitti

quienes recién comienzan, explicando como crear cosas sencillas
a partir de lo que se incluye en el CD. Tener una idea de como
usar el sistema ayudará mucho a la hora de las consultas, por
ejemplo comentar algo sobre el editor "geany", "vim", contar como
se compila con "gcc" o "make", como instalar programas
adicionales (incluido juegos) etc...

dvd escribió:A mi este cd me ira bien para trabajar en otros ordenadores. Supongo que no, pero, en el modo cdlive, hay algún problema en ejecutar y guardar archivos en un pendrive, disco extraíble, etc?


No, por suerte no hay problemas de ese tipo. Con solo conectar
un pendrive el sistema lo reconoce y abre una ventana con el contenido
de la unidad extraíble. El punto de montaje para estos dispositivos suele
ser "/media/disk".

El único tema de usar el sistema en modo Live es que tarda mucho mas tiempo
en arrancar (debido a la velocidad de las unidades de CD, la compresión de
disco y el mismo soporte óptico)... una vez instalado funciona mucho mas
rápido y liviano.

Aún así, sirve para tus planes. De hecho, yo usaba un sistema Live cuando
necesitaba usar equipos de laboratorio en la universidad, eso me evitaba
tener que instalar un montón de bibliotecas cada vez que usaba un equipo
distinto...

dvd escribió:Pd: Hugo (o cualquiera), si has buscado información por internet para hacer lo del cd... tienes algún link con un buen manual para personalizar el cdlive ubuntu?


Sí, busqué mucha información sobre como personalizar Ubuntu, pero al final
opté por utilizar dos aplicaciones que simplifican mucho el proceso. UCK [1]
es un conjunto de herramientas que me permitió configurar el idioma
predefinido en el sistema, y el programa Reconstructor [2] (el que mas
utilicé) me permitió recompilar el CD, agregar programas, configurar el
aspecto visual, entre otras cosas.

En general, el procedimiento consiste en ejecutar uno de estos programas y
especificar una imagen de CD .iso, a partir de ello personalizas el sistema
a tu gusto y listo.

[1] - http://uck.sourceforge.net/
[2] - http://reconstructor.aperantis.com/
