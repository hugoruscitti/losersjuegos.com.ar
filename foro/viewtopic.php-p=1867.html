Bueno, el mejor libro (o por lo menos, el mas recomendado) es el "Red Book", llamado tambien "OpenGL Programming Guide".

Las ultimas versiones de este libro son pagas, pero se ha liberado de manera gratuita un version 1.1 de este libro.

Si bien la version gratuita es algo vieja, no deja de ser una excelente introduccion a OpenGL. Aquel que le desee profundizar y actualizar sus conocimientos deberia comprar la ultima version del libro.

Aca les paso el link de la version gratuita:

http://www.glprogramming.com/red/
