por endaramiz » Sab Mar 28, 2009 5:09 pm

http://www.gnu.org/licenses/why-not-lgpl.es.html escribió: El Proyecto GNU tiene dos licencias principales para ser usadas en bibliotecas. Una es la GPL de GNU para bibliotecas (GNU Library GPL); la otra es la GPL de GNU ordinaria. La elección de una u otra licencia tiene mucha importancia: usar la GPL para bibliotecas permite el uso de la biblioteca en programas privativos; el uso de la GPL ordinaria para una biblioteca la hace disponible únicamente para programas libres.

Nota: GPL para bibliotecas ahora se llama GPL reducida.

La AGPL no la conocía.
