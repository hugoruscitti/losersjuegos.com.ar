Re: Propuesta para llevar programación a las escuelas!!!

por hugoruscitti » Jue Mar 28, 2013 1:52 am

¡Hola a todos!

Muchas gracias por los comentarios y probar la aplicación!!!, el intérprete de python se llama
brython, es una especie de conversor de python a javascript. Así que todo ese código
se ejecuta realmente en sus propios navegadores, no hay un servidor python/ruby o algo así...

Para el coloreado de sintáxis del editor utilicé Ace (muy parecido a codemirror) y algo de angularjs
para las pestañas del manual y la consola.

El código del editor lo pueden ver en github, esta semana voy a estar actualizado bastante el código, implementando
algunas cosas nuevas y documentando:

https://github.com/hugoruscitti/pilas-editor

Ah, y la pantalla preview del juego utiliza pilasweb, una versión alternativa de pilas-engine que estamos
escribiendo en javascript:

https://github.com/hugoruscitti/pilasweb

Cuenten que les parece, me gustaría que se sumen mas personas a programar, ¿que les parece?
Hugo Ruscitti
