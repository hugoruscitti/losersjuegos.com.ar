Ah... ya se... lo ví en otra versión de ubuntu en un equipo de casa... pasa esto, resulta
que al instalar box2D desde easy_install, no se logra compilar bien como wrapper de box2d.

Hay que instalarlo a mano, como dice acá:

http://code.google.com/p/pybox2d/wiki/B ... fromSource

En resumen, siguiendo la página anterior ejecuté estos comandos
y salió andando perfecto:

Código: Seleccionar todo
sudo apt-get install build-essential python-dev swig python-pygame
sudo apt-get install subversion
svn checkout http://pybox2d.googlecode.com/svn/trunk/ pybox2d
cd pybox2d
sudo python setup.py install --force


Igual estoy instando ubuntu desde cero, la versión maverick, para
corregir el manual y que quede perfecto...
