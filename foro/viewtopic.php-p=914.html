Saludos, quisiera aportar mi recomendación. Python se parece
en muchos aspectos a los lenguajes que ya conoces, pero aún
así es diferente, único, y no hay que tomarlo con prisa.

Creo que la mejor forma de comenzar es por un tutorial breve,
aprender a realizar cosas sencillas como generar listas, contar
elementos, abrir un archivo etc. En la siguiente página hay
muchos tutoriales interesantes para aprender sobre python:

http://www.python.com.ar/moin/AprendiendoPython

yo en particular creo que el mas simple y rápido para leer es
el siguiente:

http://usuarios.lycos.es/arturosa/pag/d ... taneo.html

se que al principio puede sonar un poco duro: pero es mejor
comenzar con el intérprete de solo texto. Es bastante interactivo
y te permite "curiosear" con mas facilidad que un entorno de
programación. Después de todo estás aprendiendo ¿no?


PD: python 3.0 es muy reciente, para usar pygame te recomiendo
usar python2.5 (de hecho el instalador de la página de pygame
está preparado para esa versión o la anterior (2.4)). Ten en
cuenta eso...
