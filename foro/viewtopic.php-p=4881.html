por pchack » Mar Dic 28, 2010 5:50 am

Saludos, les presento un nuevo actor que desarrolle para pilas este es un boton, esta bastante interesante
porque es personalizable, esto quiere decir que puedes asignarle tus propias imagenes y convertirlas en boton, tambien
tiene la capacidad de poder conectarle funciones, por ejemplo, al estar presionado o al estar sobre el y lo mejor
es que es bastante sencillo y poderoso.

Para crear un boton basta solo con escribir la siguiente sentencia:
Código: Seleccionar todo
boton = pilas.actores.Boton()


aparecera una imagen como esta:
boton_normal.png


hasta el momento no reacciona.

si queremos que al presionar cambie la apariencia a la de un boton presionado
se lograria de la siguiente manera:

Código: Seleccionar todo
def pintar_boton_presionado():
    boton.pintar_presionado() #cambia a la imagen presionada del boton
             
boton.conectar_presionado(pintar_boton_presionado)


con esto al presionar el boton se veria asi:
boton_press.png


como dije anteriormente este boton tambien se puede personalizar esto se logra indicando las rutas de
las imagenes para cada estado y luego al llamar a metodos como boton.pintar_presionado(), no se pintaria
la imagen predeterminada si no la que le señalamos.

en codigo seria asi:
Código: Seleccionar todo
# se indican las rutas en el orden de estado normal, presionado, sobre
boton = pilas.actores.Boton('imagen_normal.png', 'imagen_presionada.png', 'imagen_sobre.png' )

boton.pintar_presionado() # se veria la imagen 'imagen_presionada.png'


bueno si quieres saber sobre el funcionamiento completo del boton puedes consultar la documentacion
oficial la cual pronto estara actualizada con este boton

bueno con este boton he desarrollado un minijuego, el famoso tres en raya:
aqui una capturas:

b.png


descarga la ultima version de pilas en el repositorio oficial:

https://bitbucket.org/hugoruscitti/pilas

y en la seccion minijuegos encontraras el tres en raya
ademas podras utilizar el boton.

diviertanse con el juego y cuentenme que les parece.

Saludos

si deseas utilizar imágenes para el boton más grande que las predeterminadas, si se puede hacer.

sobre el boton, en una respuesta anterior explique un poco el funcionamiento, ademas existen algunos ejemplos.

te explico un poco el funcionamiento del boton.

lo creas asi:
Código: Seleccionar todo
boton = pilas.actores.Boton()


la particularidad de este boton es que pudes conectarle funciones, por ejemplo si quieres
que muestre un mensaje en pantalla al presionarlo.
se haria asi:

Código: Seleccionar todo
def mensaje_avisando():
    pilas.avisar("presiono boton")

boton.conectar_presionado(mensaje_avisando)


puedes conectar funciones cuando el boton se encuentre normal (sin accion), al presionarlo, o al estar sobre él con los
siguientes metodos.
Código: Seleccionar todo
boton.conectar_sobre(funcion_que_queremos)
boton.conectar_presionado(funcion_que_queremos)
boton.conectar_normal(funcion_que_queremos)

y si quieres tambien puedes desconectarlas:
Código: Seleccionar todo
boton.desconectar_presionado(funcion_que_queremos)


otra cosa interesante es que un boton no siempre se ve igual, por ejemplo, si pasamos sobre el se ve mas iluminado.

para cambiar de imagen al boton se realiza de la siguiente manera:

Código: Seleccionar todo
# para pintar imagen interna presionado
boton.pintar_presionado()

# para pintar imagen interna sobre
boton.pintar_sobre()

# para pintar imagen interna normal
boton.pintar_normal()


internamente el boton tiene tres imagenes previamente cargadas, una normal, otra presionado, y otra sobre él,
si quieres mostrar cada imagen escribes una sentencia como las anteriormente mensionada.

ahora si quieres que tu boton tenga otras apariencia.

las asignas al crearlo:

Código: Seleccionar todo
# primero las cordenadas y luego las imagenes
boton = pilas.actores.Boton( 0, 0, "imagen_normal.png", "imagen_presionado.png","imagen_sobre.png")


asi al llamar al metodo:
Código: Seleccionar todo
boton.pintar_presionado()

se veria la "imagen_presionado.png"

observa los ejemplos que trae pilas, en donde se encuentran algunos que muestran el funcionamiento
del boton, y ademas traen comentarios.

espero que te sirva esta explicación

Saludos.
