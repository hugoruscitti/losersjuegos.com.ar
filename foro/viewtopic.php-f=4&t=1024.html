por hugoruscitti » Mié Ene 26, 2011 8:57 pm

Buenas, aparentemente te falta una biblioteca que se llama "swig", sin ella no podrías
usar el módulo de física.

La linea que podrías ejecutar para que todo te funcione OK es:

Código: Seleccionar todo
sudo apt-get install python-setuptools python-sfml python-pygame swig python-elements
sudo easy_install -U pilas


Hay un artículo de instalación completo en ubuntu en la siguiente página:

http://www.pilas-engine.com.ar/doku.php ... _en_ubuntu
