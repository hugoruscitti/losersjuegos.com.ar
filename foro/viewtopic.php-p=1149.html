CVS es un programa para el control de versiones de lo que sea que estás programando (o no, porque también puedes utilizarlo para otras cosas como escribir un libro).
Para saber sus fundamentos en este mismo foro se han puesto algunas direcciones de documentación de varios programas para control de versiones (+ info en este post).
Si te interesa aprender, nos hemos juntado algunos de los usuarios de este foro para programar juegos, empezaremos con un tetris, y comenzaremos utilizando Darcs para el control de versiones. Si te apuntas eres bienvenido.
