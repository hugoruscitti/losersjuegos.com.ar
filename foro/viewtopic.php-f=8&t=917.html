Usando la biblioteca GTK podrías hacer algo así:



código:

Código: Seleccionar todo
import gtk

ventana = gtk.Window()
ventana.set_title("Titulo del calendario")

calendario = gtk.Calendar()
ventana.add(calendario)

ventana.connect('destroy', gtk.main_quit)

ventana.show_all()
gtk.main()
