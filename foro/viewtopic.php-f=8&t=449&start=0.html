hola a todos los usuarios de losersjuegos, se que este post no deberia ir aqui pero, como este es el mas visitado pues queria saber que opinan de una idea que vino a mi mente, talvez ustedes conozcan mejor del caso.

resulta que seria bueno que losersjuegos lanzara su primer curso online, hacerca de pygame como desarrollar juegos, la idea es que el usuario final al terminar el curso sepa hacer su propio juego, claro que para hacer el curso se necesitan una serie de requisitos por ejemplo yo creo que serian asi:

1) Estar registrado
2) Tener conocimientos sobre python(lenguaje de programacion).
3) Muchas ganas de aprender
4) muchas mas cosas

bueno para hacer este curso se requiere de personal y ayuda entre todos, no se creo yo que el mas indicado para dar el curso es nuestro Master Hugo.

hacer un curso de esto requiere mucha planificacion y un monton de cosas, pero tengo entendido que hay un material web que permite a estos cursos hacerlos de manera sencilla.
